# 🍳 ratatouille-server.toml
## Ratatouille美食模组 - 服务器配置翻译

---

```toml
[kinetics]  # 动力学设置

    # 这个模组与Create机械动力联动
    # 添加了一些烹饪相关的动力机器

[kinetics.stressValues.ratatouille.impact]  # 各机器的应力消耗

    # 机械脱模器
    # 用于从模具中取出食物
    # 应力消耗：8.0
    mechanical_demolder = 8.0

    # 脱粒机
    # 用于分离谷物和秸秆
    # 应力消耗：4.0
    thresher = 4.0

    # 烤箱风扇
    # 用于烤箱烹饪
    # 应力消耗：2.0
    oven_fan = 2.0

    # 涂抹器
    # 用于在食物上涂抹酱料
    # 应力消耗：2.0
    spreader = 2.0
```

---

## 📝 配置说明

### 应力消耗对比

| 机器 | 应力消耗 | 用途 |
|-----|---------|------|
| 机械脱模器 | 8.0 | 从模具取出食物 |
| 脱粒机 | 4.0 | 分离谷物 |
| 烤箱风扇 | 2.0 | 烤箱烹饪 |
| 涂抹器 | 2.0 | 涂抹酱料 |

**提示**: 这些机器都与Create的动力系统兼容，需要提供旋转动力才能工作。
