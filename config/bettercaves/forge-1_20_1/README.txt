######################################
#         liquidregions.json         #
######################################
  此文件包含每个维度的液体区域设置。
虽然 Better Caves 的雕刻器是按生物群系添加的，但液体区域
（即世界底部的巨大水池和熔岩池）是按维度处理的。
liquidregions.json 文件用于定义每个维度的液体区域。
您可以添加或删除维度，或修改现有维度的设置。
如果某个维度未在此文件中定义，则该维度将没有任何液体区域。
在这种情况下，将使用该维度各自的 noise_settings 数据包 JSON 中指定的
普通含水层。
