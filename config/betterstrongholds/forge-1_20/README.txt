#             ores.json              #
######################################

  此文件包含描述给定矿石被选中概率的方块集选择器（见下文）。
这些概率用于要塞中的藏宝室，那里有几率生成矿石堆。
有关方块集选择器的信息，请参阅此 README 的底部。

######################################
#         方块集选择器说明          #
######################################

描述一组方块状态以及每个方块状态被选中的概率。
 - entries：一个对象，其中每个条目的键是一个方块状态，每个值是该方块状态被选中的概率。
      所有概率的总和不应超过 1.0！
 - defaultBlock：用于任何剩余概率范围的方块状态。
      例如，如果所有条目概率的总和为 0.6，则
      有 0.4 的几率选中 defaultBlock。

以下是一个方块选择器示例：
"entries": {
  "minecraft:cobblestone": 0.25,
  "minecraft:air": 0.2,
  "minecraft:stone_bricks": 0.1
},
"defaultBlock": "minecraft:oak_planks"

对于每个方块，此选择器有 25% 的几率返回圆石，20% 的几率选择空气，
10% 的几率选择石砖，以及 100 - (25 + 20 + 10) = 45% 的几率选择橡木木板（因为它是默认方块）。

######################################
#         物品集选择器说明           #
######################################

描述一组物品以及每个物品被选中的概率。
工作方式与方块集选择器相同，但使用物品而不是方块状态。
