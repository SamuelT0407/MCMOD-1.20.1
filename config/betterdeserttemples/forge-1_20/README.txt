######################################
#          itemframes.json          #
######################################
  此文件包含描述物品展示框中物品概率分布的物品随机器。
物品展示框只生成在食物储藏室和军械室中。
有关物品随机器的信息，请参阅此 README 的底部。
######################################
#         物品随机器说明          #
######################################
描述一组物品以及每个物品被选中的概率。
 - entries：一个对象，其中每个条目的键是一个物品，每个值是该物品被选中的概率。
      所有概率的总和不应超过 1.0！
 - defaultItem：用于任何剩余概率范围的物品。
      例如，如果所有条目概率的总和为 0.6，则
      有 0.4 的几率选中 defaultItem。
以下是一个物品随机器示例：
"entries": {
  "minecraft:stone_axe": 0.25,
  "minecraft:shield": 0.2,
  "minecraft:air": 0.1
},
"defaultItem": "minecraft:iron_axe"
对于每个物品，此随机器有 25% 的几率返回石斧，20% 的几率选择盾牌，
10% 的几率选择空气（什么都没有），以及 100 - (25 + 20 + 10) = 45% 的几率选择铁斧（因为它是默认物品）。
