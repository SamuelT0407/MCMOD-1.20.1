# 应用能源2 是什么？

应用能源2 (Applied Energistics 2) 原创了一套独特的机制来提供物流和存储解决方案。你可以用紧凑的 ME网络 (ME Network) 替代堆满箱子的巨大仓库，但那只是冰山一角。
应用能源旨在与模组包中的其他模组协同工作，并实现自动化。你可以设置你的系统，通过单次点击，
自动合成复杂合成链中的所有前置材料（以及最终成品），或者保持特定物品的库存数量，并在需要时自动补充，
或者仅仅是在基地各处传输物品。
