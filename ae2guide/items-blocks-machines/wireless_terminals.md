# 无线终端 (Wireless Terminals)

<Row>
  <ItemImage id="wireless_terminal" scale="4" />

  <ItemImage id="wireless_crafting_terminal" scale="4" />
</Row>

无线终端是常规有线 [终端](terminals.md) 的便携式版本。它们具有与其
有线对应物完全相同的 UI，除了代替 <ItemLink id="view_cell" /> 的插槽外，它们有 [升级卡](upgrade_cards.md) 的插槽。

为了将它们与网络配对，将终端插入连接到该网络的 <ItemLink id="wireless_access_point" />
的右上角插槽中。（上面有无线终端图片且下方有箭头的插槽）

它们必须在 <ItemLink id="wireless_access_point" /> 的范围内才能运行。

它们的能量可以在 <ItemLink id="charger" /> 中充电。

# 无线终端 (Wireless Terminal)

<ItemImage id="wireless_terminal" scale="4" />

你的基本终端，现在便携了！在 <ItemLink id="wireless_access_point" /> 的范围内，
从任何地方查看和访问你的 [网络存储](../ae2-mechanics/import-export-storage.md) 的内容
并从你的 [自动合成](../ae2-mechanics/autocrafting.md) 设置中请求东西。

# 无线合成终端 (Wireless Crafting Terminal)

<ItemImage id="wireless_crafting_terminal" scale="4" />

无线合成终端类似于普通无线终端，具有所有相同的设置和部分，但增加了一个合成网格，该网格将自动
从 [网络存储](../ae2-mechanics/import-export-storage.md) 中重新填充。Shift 单击输出时要小心！

## UI (The UI)

参见 [终端](terminals.md)

无线合成终端支持以下 [升级](upgrade_cards.md)：
