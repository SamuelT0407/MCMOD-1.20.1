# 视图元件 (View Cell)

<ItemImage id="view_cell" scale="2" />

视图元件用于过滤 [终端](terminals.md) 的显示。你在 <ItemLink id="cell_workbench" /> 中对它们进行分区。

例如，假设你希望终端仅显示选择的石头建筑材料。将视图元件分区为那些
材料并将其放置在终端中，只有那些物品才会被显示。

视图元件是可叠加的，如果你有一个橡木木板的视图元件和一个圆石的视图元件，将两者放入将显示
木板和圆石。
