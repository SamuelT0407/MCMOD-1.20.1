# 空间 IO 端口 (The Spatial IO Port)

<BlockImage id="spatial_io_port" p:powered="true" scale="8" />

空间 IO 端口用于 [空间 IO](../ae2-mechanics/spatial-io.md) 以容纳 [空间元件](spatial_cells.md)
并控制空间 IO 操作。

如果你想自动化它，可以通过漏斗或 ae2 总线等任何物品物流插入和提取元件。
