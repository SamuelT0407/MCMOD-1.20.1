# 扳手 (Wrenches)

<Row>
  <ItemImage id="certus_quartz_wrench" scale="4" />

  <ItemImage id="nether_quartz_wrench" scale="4" />
</Row>

扳手用于旋转 ae2 设备（右键单击），和拆卸 ae2 方块（shift 右键单击）。
[子部件](../ae2-mechanics/cable-subparts.md) 可以从线缆上移除，而不会破坏线缆上的所有东西
（或者可以移除线缆而不会破坏子部件。）

许多 ae2 方块可以旋转，所以仅仅因为本指南没有说你可以旋转某物并不意味着你不能。
